项目介绍场景转换人物对话物品拾取
需求说明 #
  1. H2场景中新增两件物品，每件物品包含场景图、UI图、名称三要素
  2. 物品自身带有碰撞区域属性、点击事件、及点击后动画效果
数据description 描述
icon_Texture UI中图片
scene_Texture 场景中物品图片
交互点击事件
效果点击动画效果
实现路径 #

主要针对3个脚本进行操作

  1. 新建Item类（继承Resource），并在item中定义所需数据元素；创建tres资源进行继承，实例化item（key、ticket）
  2. 优化原interactable类
    1. 定义新增物品Texture，并添加setget属性方法
    2. 为新增物品自动添加CollisionShape2D碰撞区域
  3. 新建SceneItem类，继承interactable类，包括点击事件override重写、添加点击动画效果
物品数据流转 #
  1. Item类定义物品基础数据
  2. tres引用Item类，进行物品资源实例化，并赋值
  3. SceneItem继承Interactable，物品资源赋值导出变量，带入SceneItem类中，处理点击事件重写及动画效果
  4. Interactable为新增物品添加Collision碰撞区域

SceneItem类说明 #
  • 工具类，关键字tool
  • 继承Interactable，可进行交互
  • 设置类名，单独为SceneItem

export var item: Resource

Godot 中到导出变量的类型只能是

  • 基础类型：int、float、Vector2
  • 内置资源：Texture、AudioStream

无法设置自定义类的类型

可以通过setget参数进行设置 如果是item 否则设置为空

export var tex: Texture setget set_texture
  
func set_texture(v: Texture):
  tex = v
  for node in get_children():
    if node.owner == null:
      node.queue_free()
  
  if tex == null:
    return
	
  var sprite := Sprite.new()
    sprite.texture = tex
    add.child(sprite)
    
    var rect := RectangleShape2D.new()
      rect.extents = v.get_size() / 2
    
    var collider := CollisionShape2D.new()
    collider.shape = rect
    add.child(collider)
